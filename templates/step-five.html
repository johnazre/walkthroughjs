Step Five: Arrays

Simply, arrays are a list of values, separated by commas, surrounded in square brackets. There's not a whole lot more to it than that. They are incredibly useful to know how to use and even more useful to know how to search and pinpoint, especially when it comes to JSON and things like that. The major topics we will hit on are:


  • Overivew of an array

  • Looping through arrays


Overview of an array

Like we said before, an array is simply a list of values, separated by commas, and surrounded by square brackets. Your question should now be, though, how do I access a value in said array? AHA! Let's find out with the example below:


//Empty arrays are useful for a place to put values. You can declare them empty with no issues.
var emptyArray = [];

//You don't have to declare an array any differently than anything other value on the left side.
var arrayWithValues = [1, 2, 3, 4];

//Arrays can have any type of value in it, even mixed values!
var mixedArray = [1, "2", 3, function four { console.log('4')}];

//We can access a value in the array by using: nameOfTheArray[index]

//REMEMBER!!! Arrays are zero-indexed, so if you want the first value in the array, use index zero! Let's use the arrayWithValues array to pull a couple values.
arrayWithValues[1]; //Will pull the second value, 2. arrayWithValues[2]; //Will pull the third value, 3.

Let's talk about how these can be useful by touching on how to loop through arrays with a for-loop.



While Loop

While loops are useful for situations that you're not 100% sure about how much stuff you're going to have to loop through


//Let's declare an array, so we have something to loop through:
var array = [1, "apple", 3, "business"];

//Now, let's print out all of the values in the array.
for (var i = 0; i < array.length; i++){
console.log(array[i]);
}
//Will log: 1, apple, 3, and business;
