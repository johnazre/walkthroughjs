Step Five: Arrays

Simply, arrays are a list of values, separated by commas, surrounded in square brackets. There's not a whole lot more to it than that. They are incredibly useful to know how to use and even more useful to know how to search and pinpoint, especially when it comes to JSON and things like that. The major topics we will hit on are:


  • Overivew of an array

  • Looping through arrays


Overview of an array

Like we said before, an array is simply a list of values, separated by commas, and surrounded by square brackets. Your question should now be, though, how do I access a value in said array? AHA! Let's find out with the example below:


//Empty arrays are useful for a place to put values. You can declare them empty with no issues. var emptyArray = [];

//You don't have to declare an array any differently than anything other value on the left side.
var arrayWithValues = [1, 2, 3, 4];

//Arrays can have any type of value in it, even mixed values!
var mixedArray = [1, "2", 3, function four { console.log('4')}];

//We can access a value in the array by using: nameOfTheArray[index]
//REMEMBER!!! Arrays are zero-indexed, so if you want the first value in the array, use index zero!
//

Not sure just what happened there? Let's walk through it. We stated that it was going to be a for loop first. After that, within the parentheses, we declared a variable called "i", set it to equal zero, then put a semi-colon. That variable declaration is telling the loop where to start. After the first semi-colon, we have "i < 6". We are saying do whatever is in the loop while "i" is still less than 6, as soon as it becomes 6, stop the loop. After the second semi-colon, you see "i++". That is shorthand for "i = i + 1". That last part says after every time the loop finished running, add 1 to i.

Within the loop, all it says is to console.log the variable "i" every time it runs.



While Loop

While loops are useful for situations that you're not 100% sure about how much stuff you're going to have to loop through


var i = 1;
while (i < 6) {
console.log(i);
}
//Will log 1, 2, 3, 4, 5, 6;

Conclusion

Loops aren't a difficult feat to accomplish. They just take practice and experience to see why they are so useful.
