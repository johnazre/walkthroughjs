Overview of an object

Objects are pretty much the building blocks of an application, which sounds a little arbitrary. As you learn more and more about them, though, it will be come clearer. Objects are basically a very organize way to lable and store data so it is easily retreivable. It is made up of key/value pairs, separated by commas, and surrounded by curly brackets. Take a look at the example below:


var object = {
key: value,
key: value,
};
//The thing on the left is known as the "key",also known as the "property".
//The thing to the right of the colon is known as the "value", hence the "key/value pairs" name.

Creating an object

Creating objects is no different on the left side than before, however, notice in the example above that instead of square brackets, objects use curly brackets to enclose the key/value pairs, which are separated by commas. Let's do an example below.


var exampleObject = {
firstName: "Dave",
lastName: "Davis",
};

Manipulating an object

You can manipulate objects in two different ways. Bracket-notation or dot-notation. Let's continue using this example, since we have it put together already.


var exampleObject = {
firstName: "Dave",
lastName: "Davis",
};

//If we want to use dot-notation to change the firstName to James, we would use:
exampleObject['firstName'] = "James";

//If we want to do the same thing with dot-notation:
exampleObject.firstName = "James";

//If we wanted to create a new key/value pair, we could use either notation in the same way:
exampleObject.favoriteColor = "Green";
exampleObject['favoriteColor'] = "Green";
